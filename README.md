# HTTP Server for OrbitDB

HTTP server and an API to run and manage [OrbitDB](https://github.com/orbitdb/orbit-db) databases.

***Work in Progress!***

## Install
```
git clone https://github.com/haadcode/orbit-db-http-server.git
cd orbit-db-http-server/
npm install
```

## Run
```
npm start
```

## Tests
```
npm test
```

## Lint
```
npm run lint
```
## Code Coverage
```

npm run cover
```
